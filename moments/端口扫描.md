### 监听端口

  

```shell

nc -lvnp <port>

```

  

### 扫描端口

  

```shell

nc -zv[u] <port>

nmap <ip_addr> -p <port>

```

  

### 扫描活动主机

  

```shell

nmap -T<num> -sn <ip_addr>

```


### 查看端口占用情况



#### 列出所有正在监听的 TCP 和 UDP 端口，并显示对应的进程信息

  

```shell

ss -tunlp

```

  

> [!tip]

> ss 是 netstat 的现代替代品
